# OSINT News Bot

OSINT News Bot — це навчальний Python-проєкт для моніторингу новин з RSS-джерел.
Програма читає список ключових слів, перевіряє новинні RSS-стрічки та показує
новини, які можуть бути корисними для інформаційного моніторингу.

## Можливості програми

- Читає RSS-джерела з файлу `sources.txt`.
- Читає ключові слова з файлу `keywords.txt`.
- Фільтрує новини за ключовими словами.
- Групує знайдені новини за ключовими словами.
- Показує назву, джерело, посилання та короткий висновок.
- Оцінює важливість новини: `ВИСОКА`, `СЕРЕДНЯ` або `НИЗЬКА`.
- Має просте меню в терміналі.
- Може показувати тільки важливі новини.
- Дозволяє зробити тимчасовий пошук за власним ключовим словом.
- Записує інформацію про запуск у файл `logs.txt`.

## Структура файлів

```text
osint-news-bot/
├── main.py        # основний файл програми
├── keywords.txt   # ключові слова для пошуку новин
├── sources.txt    # RSS-джерела новин
├── logs.txt       # журнал запусків програми
└── README.md      # опис проєкту та інструкції
```

## Як встановити залежності

Для роботи програми потрібна бібліотека `feedparser`.

Встановіть її командою:

```bash
pip install feedparser
```

Якщо на Windows команда `pip` не працює, спробуйте:

```bash
python -m pip install feedparser
```

## Як запустити програму

Запустіть файл `main.py`:

```bash
python main.py
```

Після запуску з'явиться меню:

```text
=== OSINT News Bot ===
1. Запустити моніторинг новин
2. Показати ключові слова
3. Показати RSS-джерела
4. Вийти
5. Показати тільки важливі новини
6. Пошук новин за власним ключовим словом
```

Введіть номер потрібного пункту та натисніть `Enter`.

## Telegram Bot

Для Telegram-бота потрібні бібліотеки `python-telegram-bot` з підтримкою `job-queue` і `python-dotenv`:

```bash
pip install "python-telegram-bot[job-queue]" python-dotenv
```

1. Створити бота через `@BotFather`.
2. Отримати `TELEGRAM_BOT_TOKEN`.
3. Створити файл `.env`.
4. Додати в `.env`:

```text
TELEGRAM_BOT_TOKEN=your_bot_token_here
```

Windows PowerShell:

```powershell
$env:TELEGRAM_BOT_TOKEN="your_bot_token_here"
python telegram_bot.py
```

5. Запустити:

```bash
python telegram_bot.py
```

Важливо: не вставляйте реальний токен прямо в код.

Команди бота:

```text
/start — перевірити запуск бота
/news — показати перші 5 новин
/important — показати тільки важливі новини
/summary — показати оперативну зведену довідку
/subscribe — підписатися на OSINT-сповіщення
/unsubscribe — відписатися від OSINT-сповіщень
/status — показати статус Telegram-бота
/ping — перевірити роботу бота
/botstats — показати статистику використання бота
/settings — показати налаштування Telegram-бота
/sources — показати RSS-джерела
/keywords — показати ключові слова
/restartinfo — інструкція з перезапуску бота
/version — версія Telegram-бота
/about — інформація про OSINT News Bot
/myid — показати ваш Telegram ID
/health — перевірка системи
/logs — показати останні логи Telegram-бота
/clearlogs — очистити логи Telegram-бота
/clearstats — очистити статистику Telegram-бота
/admins — показати список адміністраторів
/adminhelp — службові команди
/help — допомога
```

Після `/start` бот показує кнопки:

```text
📰 Новини
🚨 Важливі
📊 Зведення
ℹ️ Допомога
```

## Типові помилки запуску Telegram-бота

Якщо бачите:

```text
Помилка: файл .env не знайдено
```

Створіть файл `.env` у папці проєкту.

Якщо бачите:

```text
Помилка: TELEGRAM_BOT_TOKEN не вказано
```

Перевірте, що у `.env` є рядок:

```text
TELEGRAM_BOT_TOKEN=your_bot_token_here
```

Не вставляйте реальний токен у код або README.

## Як додати адміністратора

1. Написати боту `/myid`.
2. Скопіювати свій Telegram ID.
3. Додати ID у файл `admins.txt`, один ID на рядок:

```text
123456789
```

4. Перезапустити бота.

## Як додавати ключові слова

Ключові слова зберігаються у файлі `keywords.txt`.

Кожне ключове слово потрібно писати з нового рядка:

```text
Україна
Росія
NATO
ЗСУ
санкції
```

Порожні рядки програма ігнорує.

## Як додавати RSS-джерела

RSS-джерела зберігаються у файлі `sources.txt`.

Кожне посилання потрібно писати з нового рядка:

```text
https://www.pravda.com.ua/rss/
https://rss.unian.net/site/news_ukr.rss
https://www.radiosvoboda.org/api/zrqiteuuir
```

Порожні рядки програма ігнорує.

## Логи

Після кожного запуску моніторингу програма додає запис у `logs.txt`.

Приклад запису:

```text
[2026-05-26 12:30]
Перевірено джерел: 3
Знайдено новин: 7
Статус: SUCCESS
```

Якщо під час отримання RSS була помилка, статус буде `ERROR`.

Telegram-бот записує виклики команд у `telegram_logs.txt`.
